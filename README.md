# KELASC_Kelompok3_SOLID

## Deskripsi
Repository ini berisi implementasi dan analisis prinsip SOLID menggunakan bahasa Python pada studi kasus sistem Kebun Binatang.

## Prinsip SOLID yang Dianalisis
1. Single Responsibility Principle (SRP)
2. Open Closed Principle (OCP)
3. Liskov Substitution Principle (LSP)
4. Interface Segregation Principle (ISP)
5. Dependency Inversion Principle (DIP)

## Struktur File
- `main.py` : Kode program hasil penerapan prinsip SOLID.
- `Latihan Analisis Prinsip SOLID.docx` : Dokumen analisis dan jawaban tugas.

## Cara Menjalankan Program
1. Pastikan Python telah terinstal.
2. Jalankan file `main.py`.
3. Program akan menampilkan proses perawatan hewan pada kebun binatang.
